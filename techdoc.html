Introduction to APT

Advanced Package Tool, or APT, is a free-software user interface that works with core libraries to handle the installation and removal of software on Debian, Ubuntu, and related Linux distributions. APT simplifies the process of managing software on Unix-like computer systems by automating the retrieval, configuration and installation of software packages, either from precompiled files or by compiling source code.

Update repositories information

APT is a very powerful tool which can help you not only install packages but also keeping them all up to date with very simple commands.

For APT to know what the latest versions of each of the packages are available, it has to retrieve the latest packages information from each of the repositories setup for APT. To accomplish this, here's the command:

$ sudo apt update

Install a package

If you know the name of the package, you can easily install it using the command below:

$ sudo apt install <package_name>

Just replace the <package_name> with your desired package. Suppose I wanted to install Pinta image editor. All I’d need to do is use the command below:

$ sudo apt install pinta

Upgrade available packages

Once you have updated the package database, you can upgrade the installed packages. The most convenient way is to upgrade all the packages that have updates available. You can use the command below for this purpose:

$ sudo apt upgrade

To upgrade only a specific program, use the command below:

sudo apt upgrade <package_name>

List installed packages

There might be times when you want to upgrade an already installed package but you may not remember its name. For that, once can make a list of all installed packages to review them. To make this list, run this command:

$ apt list --installed

The output will show the list of installed packages (including the ones features by the system distribution) sorted alphabetically. Each line item will show the package name and its version.

Uninstall a package

Installing packages isn’t the only thing you can do with apt. You can also remove packages with it. All you need to do is to use the command in this manner:

$ sudo apt remove <package_name>

Auto-completion works here as well. So just start typing package name and press tab, and it will suggest all the installed packages starting with those letters.
